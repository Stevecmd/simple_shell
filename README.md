# simple_shell
This is my repo for 0x16. C - Simple Shell

Executing the individual files: <br/>

Main entry point:
`gcc -o prompt prompt.c`
`./prompt`

Get process ID:
`gcc -o pid pid.c`
`./pid`


`gcc -o main main.c`
`./main`

Print the environment variables:
`gcc -o env-main env-main.c`
`./env-main`
